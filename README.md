# Kubernetes Portfolio

This repository contains hands-on Kubernetes projects as part of preparation for the Certified Kubernetes Administrator (CKA) exam. 

Projects included:
1. *Kubernetes Application Deployment* – deploying nginx with Deployments, Services, ConfigMaps, Secrets, and health probes.
2. *Kubernetes Security & RBAC* – controlling access using Roles and RoleBindings.
3. *Cloud-integrated Kubernetes Project* – using AWS to deploy and manage Kubernetes workloads.
